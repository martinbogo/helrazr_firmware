HelRazr Web Flasher

Install custom firmware on your Heltec devices directly from your browser.

Heltec T114 (nRF52)

The T114 uses a simple drag-and-drop UF2 bootloader. No browser flashing tool is required.

  1. Download your helrazr-t114.uf2 file.
  2. Connect your T114 via USB.
  3. Double-click the reset button quickly to enter bootloader mode.
  4. A new USB drive will appear on your computer (usually named NRF52BOOT or similar).
  5. Drag and drop the .uf2 file onto the drive.
  6. The device will automatically reboot with the new firmware.

Heltec V3 (ESP32-S3)

Use this tool to securely flash the custom factory.bin over WebSerial.

Ensure you select the helrazr-heltec_v3.factory.bin.

Recommended when switching from Meshtastic.
